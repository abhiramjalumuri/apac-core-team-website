Project 1: Nimbus

Nimbus is a next-generation WiFi planning platform that helps visualize network coverage for homes and offices. The platform generates interactive 2D and 3D layouts and suggests the best placement of WiFi extenders (pods) to achieve optimal coverage.

Key Objectives

  • AI-assisted suggestions for WiFi extender placement/li>
  • Interactive 2D and 3D visualization of home or office layouts
  • Visual indicators for areas with weak or limited coverage

Impact

Built interactive 2D and 3D property models and provided clear guidance on WiFi extender placement. Enabled users to quickly identify weak signal areas and plan network setup effectively.

Project 2: Aurora

Aurora is a next-generation ISP platform designed to deliver feature-parity services across multiple device SDKs such as PRPL, RDK, and OpenWrt. The platform ensures consistent integration for ODMs and CSPs, while enabling scalable, cloud-based service delivery.

Key Features

  • Lightweight agent for PRPL, RDK, and OpenWrt SDKs
  • Support for multiple northbound cloud protocols
  • BUS and high-level APIs for consistent communication across firmware apps
  • Strict compliance with TR-181 schema and SDK databases

Impact

Lays the foundation for consistent services across multiple device platforms and simplifies integration for partners. Provides a scalable platform for future cloud-based service delivery.
